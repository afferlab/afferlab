# Strategy Types

Looma currently has two practical strategy categories.

## Built-In Strategies

Built-in strategies are seeded by the host and stored in the strategies table.

Today the default built-ins are:

- `Base`
- `Cloud`

`Base` is the minimal chat strategy.

`Cloud` enables Memory Cloud behavior and pulls pinned memory assets into the prompt.

## Dev Strategies

Dev strategies are user-authored TypeScript files compiled and saved through the strategy development flow.

They are stored as strategy records with `source: 'dev'` and loaded from generated bundle files under the app's user-data area.

## What Is Not There Yet

There is not yet a separate published strategy SDK package or a separate community strategy registry in this repository.
