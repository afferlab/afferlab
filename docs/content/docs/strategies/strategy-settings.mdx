# Strategy Settings

The Strategy settings page is the management surface for available strategies.

## What You Can Do Today

The current UI and backend support:

- listing strategies
- enabling and disabling strategies
- editing strategy parameter overrides
- viewing usage counts
- uninstalling strategies
- reassigning conversations when disabling or uninstalling a strategy

## Config Values

If a strategy exposes `configSchema`, Looma can show a configuration dialog and persist the chosen values.

Those values are later exposed to the strategy through `ctx.config`.

## Conversation Switching

Changing the active strategy for a conversation is handled separately from the settings list itself.

When a switch occurs, the host updates the conversation strategy, updates strategy session tracking, and may schedule replay or background reindexing depending on mode.
