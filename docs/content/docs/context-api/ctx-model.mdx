# ctx.model

`ctx.model` is the small identity object for the current turn model.

## Current Fields

Today it contains:

- `id`
- `provider`

## Usage

This is useful for:

- branching on provider-specific behavior
- tagging logs or state entries
- recording model-specific strategy behavior

The object is informational. It is not a direct model client.
