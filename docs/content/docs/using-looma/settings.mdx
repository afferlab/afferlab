# Settings

Looma's settings area is split into domain-specific sections rather than one flat preferences form.

## Current Sections

The current app includes settings areas for:

- models
- strategies
- privacy
- general app behavior
- web search

## Models

Model settings let you:

- configure provider credentials and hosts
- enable or disable models
- test provider/model connectivity
- add custom provider models
- override per-model generation settings

## Strategies

Strategy settings let you manage available strategies and edit strategy parameter overrides.

## App Settings

General settings currently include items such as theme mode, launch behavior, and auto-scroll behavior.
