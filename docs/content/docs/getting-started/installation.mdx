#

## Download

AfferLab is distributed as a desktop application. You can download AfferLab from:

- 🌐 **Official website**
https://afferlab.com

- 🐙 **GitHub releases**
https://github.com/victor-YT/afferlab/releases

---

## Install

After downloading the installer:

1. Open the downloaded file.
2. Follow the installation instructions for your operating system.
3. Launch AfferLab.
